## 1、项目介绍

基于springboot的单科成绩管理系统功能简单，实现了教师对学生管理、成绩管理功能以及学生查看自己的成绩，适合学习技术


## 2、项目技术

后端框架：springboot、mybatis

前端技术：Layui、html、css、JavaScript、JQuery

## 3、开发环境

- JAVA版本：JDK1.8
- IDE类型：IDEA、Eclipse都可运行
- 数据库类型：MySql（5.x和8.x版本都可） 
- maven版本：无限制
- 硬件环境：Windows 或者 Mac OS
